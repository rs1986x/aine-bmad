### Gap-closing batch decisions (Fast path)

- **UI system:** From-scratch lightweight CSS design tokens (no UI library). DESIGN.md carries a full token set. Matches plain React+TS in addendum; best learning value.
- **Visual direction:** Calm, minimal, neutral surface with a single accent color for primary actions and active/selected state.
- **Theme:** Light mode only (keeps scope simple). Dark mode noted as a clean future extension.
- **List ordering (resolves PRD Open Question 1):** Active Todos on top, Completed Todos sink to the bottom; within each group, newest first (by `created_at`).
- **Delete behavior:** Lightweight confirmation (dialog) before deleting, since deletion is permanent with no undo (PRD §5 / FR-5). Undo-toast option explicitly declined to avoid expanding PRD scope.
